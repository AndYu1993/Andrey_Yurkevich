Блок навигации

  • Шир. 15em
  • Цвет фона
  • Располагается слева
  • Блок навигации заполниет контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показанно в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделать отступ 2em, при помощи св-ва margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседней равным .2em, при помощи совйства margin-
bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри
данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется контентом
